## Conventions Established
- Dagster asset naming: **noun-form** (`gold_observations`, `bronze_cold_archive`, `signal_snapshot_writer`) — not verb-form
- Collection asset keys: `collect_{source_id}` where source_id is the slug from `source_catalog`
- CFTC source_id: `cftc_cot` (not yet in migration — Phase 1 addition)
- `forge_compute` = Dagster assets inside `empire_dagster_code`, not a separate container
- GE suite naming: `bronze_core` (universal) + `bronze_{source_id}` (per-adapter)
- `marts.signals_history` = `forge_compute` Python asset (not dbt)
- `signal_eligible` = `instruments.collection_tier` column (not a metric_catalog flag)
- Decision gates (DG-*) = formal decision points that block phase build prompts
